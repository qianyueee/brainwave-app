# 脳特性 6 指标 算法说明

本文档总结「脳特性チャート」6 个指标的计算方式，与代码 `lib/brain-profile.ts` 保持一致。**代码是唯一准确来源**；本文数值对应 `INDICATOR_CONFIG` 当前的暂定标定值。

---

## 1. 输入数据

每次测量是一串 **1 Hz（每秒一条）** 的样本，来源有二，字段结构相同：

- **上传文件**：BrainLink 导出的 Excel/CSV（`parseEegFile`）。
- **实时 mindmap**：头环经桥接实时采集，测量结束时用 `eegRowsFromSamples` 转成同一结构。

每条样本（`EegRow`）包含：

| 字段 | 含义 | 范围/性质 |
|---|---|---|
| `attention` | 集中度（NeuroSky eSense） | 0–100 |
| `relaxation` | 放松度（= mindmap 的 `meditation`，NeuroSky eSense） | 0–100 |
| `delta / theta` | δ / θ 频段功率 | 无量纲大数 |
| `lowAlpha / highAlpha` | 低/高 α 频段功率 | 无量纲大数 |
| `lowBeta / highBeta` | 低/高 β 频段功率 | 无量纲大数 |
| `lowGamma / highGamma` | 低/中 γ 频段功率 | 无量纲大数 |

> **要点**：`attention` / `relaxation` 是 NeuroSky **eSense 专有算法**的输出（0–100），不是原始脑电；频段功率是**无量纲的大数**，因此凡是用到频段的地方**一律先归一化成比例**，绝不用绝对值。

---

## 2. 预处理

计算前统一做一步 `trimPoorSignalEdges`：

- **裁掉首尾**连续的坏信号秒（`attention ≤ 0 且 relaxation ≤ 0`）。
- **保留中间的空档**——因为计时类指标（②⑤）把数组下标当作「第几秒」，中间不能压缩，否则时间轴会错位。
- 上传路径在解析时已裁一次；实时测量从点「測定開始」那一刻就开始（电极还没贴稳），所以两条路径都要裁，保证评分口径一致。

两类指标对坏信号的处理不同：

- **计时类（②⑤）**：用**全部**（裁剪后的）行，下标 ≈ 秒。
- **数值统计类（①③④⑥）**：进一步用 `isValidSample`（`attention > 0 或 relaxation > 0`）排除坏信号秒。

---

## 3. 通用工具函数

| 函数 | 定义 |
|---|---|
| `topPercentAvg(values, p%)` | 降序排序后取前 `ceil(n·p/100)`（至少 1）个求平均——**稳健的「峰值能力」**，避免单点噪声。 |
| `speedScore(seconds, τ)` | `clamp(round(100·τ/(τ+seconds)), 0, 100)`。越快分越高：**0 秒→100，τ 秒→50**。 |
| `runsAbove(values, thr)` | 返回每段「连续 ≥ 阈值」的长度（秒）列表。 |
| `firstSustainedCrossing(values, thr, k)` | 首次「连续 k 秒 ≥ 阈值」的**起始下标**（= i−k+1）；没有则 `null`。 |
| `findSettlingIndex(rows)` | 首次「连续 k 秒满足 `relaxation ≥ restThreshold` **且** α功率 ≥ β功率」的起始下标（**入定点**）；没有则 `null`。α = 低α+高α，β = 低β+高β。 |
| `relaxBandRatio(r, excludeDelta)` | `(低α+高α+θ) / (全频段之和)`。`excludeDelta=true` 时分母**不含 δ**（δ 易被眨眼/噪声/困倦放大，默认排除以看「清醒放松」）。 |
| `coefficientOfVariation(v)` | 变动系数 CV = 总体标准差 / 均值（均值为 0 时取 0）。 |

---

## 4. 六个指标

所有指标输出为 **0–100 的整数**，不做事后再缩放。

### ① 集中の強さ（focusIntensity）—— 集中的「深度/峰值」
- 取有效秒的 `attention`，算**前 10% 的平均**。
- `score = clamp(round(topPercentAvg(attention, 10%)), 0, 100)`
- 参数：`intensityTopPct = 10`

### ② 集中の速度（focusSpeed）—— 多快进入集中
- 在全时间轴上找 `attention` **首次连续 3 秒 ≥ 50** 的起始秒 `t`。
- `score = t===null ? 0 : speedScore(t, τ=60)`（越早进入分越高）
- 参数：`threshold = 50`，`sustainSecs = 3`，`speedTau = 60`

### ③ 集中の持続度（sustainedFocus）—— 是否「成线」地持续
- 在（裁剪后的）行上取 `attention`，找所有 ≥ 40 的连续片段 `runs`。
- **覆盖率** `coverage = (Σruns / 总秒数) × 100`
- **最长持续** `longestScore = min(100, 最长片段 / 60 × 100)`
- `score = clamp(round(0.5·coverage + 0.5·longestScore), 0, 100)`
- 参数：`threshold = 40`，`longestRefSecs = 60`，`coverWeight = 0.5`，`longestWeight = 0.5`
- 说明：此项直接用裁剪后全行，中间坏信号秒（attention=0）会被算作「低于阈值」，从而拉低覆盖率。

### ④ リラックスの深さ（relaxationDepth）—— 放松的深度
- **主观层** `level = topPercentAvg(relaxation, 10%)`（放松度前 10% 均值）
- **频段层** `bandScore = min(100, mean(relaxBandRatio) / 0.7 × 100)`，其中 `relaxBandRatio = (低α+高α+θ)/非δ全频段`
- `score = clamp(round(0.6·level + 0.4·bandScore), 0, 100)`
- 参数：`levelTopPct = 10`，`bandRef = 0.7`，`levelWeight = 0.6`，`bandWeight = 0.4`，`excludeDelta = true`

### ⑤ 入定の速度（calmnessSpeed）—— 多快切换到平静态
- 找**入定点** `t = findSettlingIndex(rows)`：首次连续 3 秒满足 `relaxation ≥ 60` 且 α ≥ β。
- `score = t===null ? 0 : speedScore(t, τ=60)`
- 参数：`restThreshold = 60`，`sustainSecs = 3`，`speedTau = 60`

### ⑥ 平静の持続度（calmnessStability）—— 平静态是否稳定
- 从入定点起（`findSettlingIndex`；若无入定点则用全程）取有效秒的 `relaxation`（至少 2 个）。
- 算变动系数 `cv`，做两点线性映射：**CV ≤ 0.1 → 100，CV ≥ 0.4 → 0**。
- `score = clamp(round((0.4 − cv)/(0.4 − 0.1) × 100), 0, 100)`
- 参数：`cvGood = 0.1`，`cvCap = 0.4`（越不波动分越高）

---

## 5. 参数总表（`INDICATOR_CONFIG`）

| 指标组 | 参数 | 值 |
|---|---|---|
| ①② focus | `intensityTopPct` / `threshold` / `sustainSecs` / `speedTau` | 10 / 50 / 3 / 60 |
| ③ sustain | `threshold` / `longestRefSecs` / `coverWeight` / `longestWeight` | 40 / 60 / 0.5 / 0.5 |
| ④ relax | `levelTopPct` / `bandRef` / `levelWeight` / `bandWeight` / `excludeDelta` | 10 / 0.7 / 0.6 / 0.4 / true |
| ⑤ settle | `restThreshold` / `sustainSecs` / `speedTau` | 60 / 3 / 60 |
| ⑥ stable | `cvGood` / `cvCap` | 0.1 / 0.4 |

**指标 ↔ 主要输入**：
- ① 峰值 attention　② attention 首达时间　③ attention 持续片段
- ④ relaxation + α/θ 频段比　⑤ 入定时间（relaxation + α≥β）　⑥ 入定后 relaxation 的 CV

**総合分**：`compositeScore = round(六项的算术平均)`（`lib/brain-measurements.ts`）。

---

## 6. 重要说明与局限

1. **标定是暂定的**：默认参数只基于实测 4 组数据（Jun/L/MS/M）的分布粗调，样本变多后需重新标定阈值。
2. **上游 Attention/Meditation 是黑盒**：它们由 NeuroSky eSense 专有算法产出，公式不公开；且 Attention 与 Meditation 天然反相关。我们的 6 指标继承了这些特性。
3. **频段务必归一化**：原始频段功率是无量纲大数，所有涉及频段的计算（④的 bandScore、⑤的 α≥β、⑥前置的入定判定）都用比例，绝对值无意义。
4. **δ 波默认排除**（④的分母）：δ 易被眨眼/噪声/困倦放大，排除后看的是「清醒放松」。
5. **计时假设 1 Hz**：②⑤把数组下标当秒，依赖每秒一条样本，且中间空档需保留。
6. **测量时若在放程序**：程序播放时 γ 频段被人为放大（`programGammaGain`，为可视化 40Hz 效果），这会写入测量数据，因此：
   - **④ 会略微下降**：γ 变大 → `relaxBandRatio` 的分母变大 → 频段层 `bandScore` 略降。
   - **①②③⑤⑥ 不受影响**：它们基于 attention/relaxation 或 α/β，与 γ 无关。
   - 若希望脑特性完全不受程序影响，应在**不放程序**时测量。
7. **无事后缩放**：6 项都是直接从原始信号算出的 0–100，没有相对排名或归一到人群分布。这既是优点（可解释、可复现），也意味着分数的「绝对高低」取决于上面这些阈值的标定。
